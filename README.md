# ws-ui

目录结构

```
ws-ui                   （WS-UI根目录）
    |--src              （核心样式代码）
    |   |--ws-ui.css    （WS-UI的主样式文件）
    |   |--common.css   （公共样式文件）
    |   |--reset.css    （CSS-Reset文件）
    |   |--layout.css   （组件文件：图标样式）
    |   |--...          （其他组件文件）
    |
    |--demo             （演示文件目录）
        |--layout.html  （布局样式文件）
        |--...          （其他组件的样式文件）
```
